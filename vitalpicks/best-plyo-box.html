FAQ

What height plyo box should I use?

Start with 20". Progress to 24" when you can comfortably jump 10+ reps. Advanced athletes use 30"+.

Is a soft or hard plyo box better?

Soft (foam-topped) is easier on joints but less durable. Hard (wood) lasts longer but requires better technique.

Can I use a plyo box for step-ups?

Yes! Many exercises: box jumps, step-ups, box squats, dips. It's a versatile tool.

How much weight can a plyo box handle?

Quality boxes handle 300+ lbs easily. Check manufacturer specs for your specific model.
